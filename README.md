p2p4j
==================================
A Java Library for P2P NAT Traversal in UDP and TCP

### 目标
- 检测NAT类型,NAT穿透建立p2p连接
- 支持UDP/TCP
- 提供一致的连接API

### 快速开始
- 启动服务端p2p4js-1和p2p4js-2
- 启动客户端p2p4jc-1和p2p4jc-2
